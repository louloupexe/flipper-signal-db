📬 Submit a Signal

Have a signal captured on your Flipper Zero? Share it with the community by opening a Pull Request on GitHub. It only takes a few minutes.

1

Fork the repo

Click Fork on GitHub to get your own copy of flipper-signal-db.

2

Add your .sub file

Copy your Flipper .sub file into the signals/ folder. Use a descriptive filename like my_gate_433.sub.

3

Update signals.json

Add a new entry to signals.json with all fields filled in (see format below).

4

Open a Pull Request

Submit a PR against main. A maintainer will review and merge it within a few days.

{
  "name":        "My Garage Door 433MHz",
  "frequency":   "433.92 MHz",
  "protocol":    "OOK",
  "description": "Single-channel remote for a XYZ brand gate opener.",
  "category":    "garage_doors",
  "contributor": "your_github_username",
  "filename":    "my_garage_door_433.sub"
}
⬆️ Open a Pull Request All signal types welcome — garage doors, car keys, intercoms, weather stations, or anything custom.
