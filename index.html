          signals.json format
          
        

        
{
  "name":        "My Garage Door 433MHz",
  "frequency":   "433.92 MHz",
  "protocol":    "OOK",
  "description": "Single-channel remote for a XYZ brand gate opener.",
  "category":    "garage_doors",
  "contributor": "your_github_username",
  "filename":    "my_garage_door_433.sub"
}

      


      

        
✅ Include accurate frequency and protocol

        
✅ Use a descriptive filename (brand_model_freq.sub)

        
✅ Write a clear description of the device and signal

        
✅ Only submit signals you own or have permission to capture

        
❌ No rolling-code signals that compromise real security

        
❌ No personally identifiable information in files
